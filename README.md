# Fish It

Versión simple en HTML/CSS/JS del juego "Fish It" (clon inspirado en el video provisto).

Archivos principales:
- `index.html` — página principal y UI
- `style.css` — estilos
- `main.js` — lógica del juego (canvas)

Cómo ejecutar:
1. Abrir `index.html` en el navegador (doble-click). Para funciones consistentes en todos los navegadores, sirve con un servidor local.

Ejemplo con Python 3 (desde la carpeta del proyecto):
```bash
python -m http.server 8000
# luego abrir http://localhost:8000
```

Controles:
- Mover ratón / arrastrar touch: mueve el cebo horizontalmente.
- Click / tocar: baja/sube el cebo; también intenta enganchar peces cercanos.
- Botones: `Start`, `Pause`, `Reset`.

Qué falta / próximas mejoras:
- Gráficos y sonidos más pulidos
- Animaciones específicas por especie
- Guardado de puntuaciones

Si quieres que adapte la jugabilidad para que coincida exactamente con el video, indícame los detalles visuales o mecánicas que quieras replicar y los implementaré.
